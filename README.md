# 🌐 Frontend Extensions & Mini Projects Collection

This repository contains a collection of my frontend mini projects and Chrome extensions built using HTML, CSS, and JavaScript.

These projects were created to strengthen my core frontend development skills, practice API integration, and understand browser extension development.

---

## 🚀 Projects Included

### 1️⃣ Chuck Norris Chrome Extension 😎
A Chrome extension that fetches and displays random Chuck Norris jokes using a public API.

**Features:**
- Fetch API integration
- Dynamic DOM updates
- Chrome Extension structure

---

### 2️⃣ Chuck Norris Joke Generator Web App 😂
A simple web application that generates random Chuck Norris jokes.

**Features:**
- API integration
- Interactive button
- Clean UI design

---

### 3️⃣ Habit Tracker Web App 📅
An interactive habit tracking application to monitor daily habits and maintain streaks.

**Features:**
- 31-day clickable calendar
- Habit selection dropdown
- Streak tracking section
- Reminder option

---

## ⚙️ How to Use

1. Clone this repository:
   ```
   git clone <your-repo-link>
   ```
2. Open any project folder.
3. Open the HTML file in your browser.
4. For Chrome Extensions:
   - Go to `chrome://extensions/`
   - Enable Developer Mode
   - Click **Load Unpacked**
   - Select the extension folder
