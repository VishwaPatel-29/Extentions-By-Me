# 📅 Habit Tracker Web App

A simple and interactive Habit Tracker web application that helps users build and maintain daily habits by tracking progress throughout the month.

This project is built using HTML, CSS, and JavaScript and focuses on DOM manipulation and user interaction.

---

## 🚀 Features

- Add and select habits from dropdown list
- 31-day calendar tracking system
- Clickable day buttons
- Streak indicator section 🔥
- Reminder checkbox option
- Clean and simple UI
- Beginner-friendly frontend project

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

## ⚙️ How to Run This Project

1. Download or clone this repository.
2. Open the project folder.
3. Open `Habit Tracker.html` in your browser.
4. Select a habit from the dropdown.
5. Click on the calendar days to track your progress.

---

## 📌 How It Works

- Users select a habit from the dropdown menu.
- Each day of the month is represented by a clickable button.
- Clicking a day button marks habit completion.
- The streak section shows consistency tracking.
- Reminder checkbox allows enabling habit reminder (if implemented in JS).
