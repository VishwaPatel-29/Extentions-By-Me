# Chuck Norris Chrome Extension 😎

A simple Chrome Extension that displays a random Chuck Norris joke using the Chuck Norris API.

This is my first Chrome Extension project built using HTML, CSS, and JavaScript.

---

## 🚀 Features

- Displays a random Chuck Norris joke
- Fetches data from a public API
- Simple and clean user interface
- Interactive buttons (Start, End, API)
- Beginner-friendly project

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Chuck Norris API (https://api.chucknorris.io)

## ⚙️ How to Install This Extension in Chrome

1. Download or clone this repository.
2. Open Google Chrome.
3. Go to: chrome://extensions/
4. Turn ON "Developer Mode" (top right corner).
5. Click on "Load unpacked".
6. Select the project folder.
7. The extension will be installed and ready to use.

---

## 📌 How It Works

- When the **API** button is clicked:
  - A fetch request is sent to:
    https://api.chucknorris.io/jokes/random
  - The response is received in JSON format.
  - The joke text is displayed inside the paragraph tag.
  - The Chuck Norris image is shown.
